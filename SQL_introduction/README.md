# SQL - Introduction

A collection of SQL scripts covering the basics of MySQL — databases, tables, and queries — as part of the ALU Higher Level Programming curriculum.

## Learning Objectives

By the end of this project, you should be able to explain:

- What a database and relational database are
- What SQL and MySQL are
- How to create and delete a database
- What DDL and DML stand for
- How to CREATE or ALTER a table
- How to SELECT, INSERT, UPDATE, and DELETE data
- What subqueries are and how to use MySQL functions

## Requirements

- MySQL 8.0 (Ubuntu 20.04 LTS)
- All SQL keywords must be uppercase
- Every query must have a comment just before it
- Every file must start with a comment describing the task

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | `0-list_databases.sql` | List all databases |
| 1 | `1-create_database_if_missing.sql` | Create database `hbtn_0c_0` if it doesn't exist |
| 2 | `2-remove_database.sql` | Drop database `hbtn_0c_0` if it exists |
| 3 | `3-list_tables.sql` | List all tables in a database |
| 4 | `4-first_table.sql` | Create table `first_table` |
| 5 | `5-full_table.sql` | Print full description of `first_table` |
| 6 | `6-list_values.sql` | List all rows of `first_table` |
| 7 | `7-insert_value.sql` | Insert a row into `first_table` |
| 8 | `8-count_89.sql` | Count records with id = 89 |
| 9 | `9-full_creation.sql` | Create `second_table` and insert 4 records |
| 10 | `10-top_score.sql` | List all records ordered by score |
| 11 | `11-best_score.sql` | List records with score >= 10 |
| 12 | `12-no_cheating.sql` | Update Bob's score to 10 by name |
| 13 | `13-change_class.sql` | Delete records with score <= 5 |
| 14 | `14-average.sql` | Compute average score |
| 15 | `15-groups.sql` | Count records per score |
| 16 | `16-no_link.sql` | List records that have a name value |
