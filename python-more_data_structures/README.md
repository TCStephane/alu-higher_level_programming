# Python - More Data Structures: Set, Dictionary

## Description

This project explores Python's more advanced built-in data structures — sets and dictionaries — alongside functional programming tools like `lambda`, `map`, `filter`, and `reduce`. All tasks are implemented in Python 3 without importing any external modules.

## Learning Objectives

By the end of this project, you should be able to explain:

- Why Python programming is awesome
- What sets are and how to use them
- The most common set methods and when to use sets versus lists
- How to iterate over a set
- What dictionaries are, how to use them, and when to prefer them over lists or sets
- What a key is in a dictionary and how to iterate over a dictionary
- What a lambda function is
- What `map`, `reduce`, and `filter` functions are and how to use them

## Requirements

- Python 3.8.5 (Ubuntu 20.04 LTS)
- All files must be executable and start with `#!/usr/bin/python3`
- Code must follow `pycodestyle` (version 2.7.\*)
- No module imports allowed (unless stated otherwise)

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | `0-square_matrix_simple.py` | Returns a new matrix with each value squared |
| 1 | `1-search_replace.py` | Replaces all occurrences of an element in a new list |
| 2 | `2-uniq_add.py` | Sums all unique integers in a list |
| 3 | `3-common_elements.py` | Returns a set of common elements between two sets |
| 4 | `4-only_diff_elements.py` | Returns elements present in only one of two sets |
| 5 | `5-number_keys.py` | Returns the number of keys in a dictionary |
| 6 | `6-print_sorted_dictionary.py` | Prints a dictionary sorted by keys |
| 7 | `7-update_dictionary.py` | Adds or updates a key/value pair in a dictionary |
| 8 | `8-simple_delete.py` | Deletes a key from a dictionary |
| 9 | `9-multiply_by_2.py` | Returns a new dictionary with all values multiplied by 2 |
| 10 | `10-best_score.py` | Returns the key with the highest integer value |
| 11 | `11-multiply_list_map.py` | Multiplies all list values by a number using `map` |
| 12 | `12-roman_to_int.py` | Converts a Roman numeral string to an integer |
