Python - Data Structures: Lists, Tuples
A collection of Python functions exploring core data structure concepts — lists, tuples, and sequences — as part of the ALU Higher Level Programming curriculum.
Learning Objectives
By the end of this project, you should be able to explain:

What lists are and how to use them
The differences and similarities between strings and lists
The most common list methods and how to use them
How to use lists as stacks and queues
What list comprehensions are and how to use them
What tuples are, how to use them, and when to prefer them over lists
What sequences, tuple packing, and sequence unpacking are
What the del statement does and how to use it

Requirements

Python 3.8.5 (Ubuntu 20.04 LTS)
pycodestyle 2.7.*
No external modules allowed (unless stated otherwise)
All files must be executable and end with a new line
First line of every file: #!/usr/bin/python3
