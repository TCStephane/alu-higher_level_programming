# Python - Input/Output

A collection of Python functions and scripts covering file handling and JSON serialization/deserialization, as part of the ALU Higher Level Programming curriculum.

## Learning Objectives

By the end of this project, you should be able to explain:

- How to open, read, write, and append to files
- How to read a file line by line and move the cursor
- How to make sure a file is closed after using it
- What the `with` statement is and how to use it
- What JSON is, and what serialization and deserialization mean
- How to convert Python data structures to JSON strings and back

## Requirements

- Python 3.8.5 (Ubuntu 20.04 LTS)
- pycodestyle 2.7.*
- No external modules unless stated otherwise
- All files must be executable and end with a new line
- First line of every file: `#!/usr/bin/python3`

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | `0-read_file.py` | Read a UTF-8 text file and print it to stdout |
| 1 | `1-write_file.py` | Write a string to a file, returns number of characters written |
| 2 | `2-append_write.py` | Append a string to a file, returns number of characters added |
| 3 | `3-to_json_string.py` | Return the JSON string representation of an object |
| 4 | `4-from_json_string.py` | Return a Python object from a JSON string |
| 5 | `5-save_to_json_file.py` | Write an object to a file using JSON representation |
| 6 | `6-load_from_json_file.py` | Create a Python object from a JSON file |
| 7 | `7-add_item.py` | Script that adds command-line arguments to a JSON list file |
| 8 | `8-class_to_json.py` | Return dictionary description of an object for JSON serialization |
| 9 | `9-student.py` | `Student` class with `to_json()` method |
| 10 | `10-student.py` | `Student` class with filtered `to_json(attrs)` method |
| 11 | `11-student.py` | `Student` class with `to_json()` and `reload_from_json()` methods |
| 12 | `12-pascal_triangle.py` | Return Pascal's triangle of `n` as a list of lists |
